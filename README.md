# Highcharts License Calculator

An interactive calculator for Highcharts license pricing with automatic bundle discounts.

## Features

- Interactive product selection
- Automatic bundle pricing when Core is included
- Real-time price calculations
- Clean, modern UI with Tailwind CSS

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

This project is automatically deployed to GitHub Pages when changes are pushed to the `main` branch.

Live site: https://jameshighcharts.github.io/standalone-chart-test/

## Technology Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS
- GitHub Pages
